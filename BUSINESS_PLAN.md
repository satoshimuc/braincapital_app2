# UNLOCK ブランド概要 / 事業計画

## 1. ブランド概要

### ブランド名

UNLOCK

### ミッション

**Unlock Potential**

人・物・事が持つ
まだ開かれていない可能性を解放する。

### ブランドコンセプト

UNLOCKは
**遊び・思考・身体・創造**を通して
人の可能性を解放するブランドである。

### コアメソッド

Play → Brain → Body → Create

1. **Play（遊び）** — 好奇心から始まる体験
2. **Brain（思考）** — 発見・理解・判断
3. **Body（身体）** — 行動・挑戦
4. **Create（創造）** — 新しい価値や可能性の創出

---

## 2. ブランドビジョン

世界中の人々が
自分の可能性に気づき
それを発揮できる社会をつくる。

UNLOCKは
**可能性を解放するプラットフォームブランド**になる。

---

## 3. ターゲット領域

UNLOCKは以下5つの領域で展開する。

### 3-1. 子ども（Kids）

| 項目 | 内容 |
|------|------|
| **目的** | 可能性の土台を育てる |
| **提供価値** | 遊びながら学ぶ、創造力、思考力、挑戦する力 |
| **コンテンツ** | LUMOキャラクター、絵本、学習コンテンツ、UNLOCK BOX、ワークショップ |

### 3-2. 法人（Corporate）

| 項目 | 内容 |
|------|------|
| **目的** | 組織の可能性を解放 |
| **提供価値** | チームビルディング、創造的思考、問題解決、イノベーション |
| **サービス** | 企業研修、ワークショップ、チームトレーニング |

### 3-3. プロフェッショナル（Professional）

| 項目 | 内容 |
|------|------|
| **目的** | 個人パフォーマンスの最大化 |
| **対象** | 経営者、クリエイター、専門職、リーダー |
| **サービス** | 脳トレーニング、思考トレーニング、パフォーマンスコーチング |

### 3-4. スポーツチーム（Sports）

| 項目 | 内容 |
|------|------|
| **目的** | パフォーマンス向上 |
| **提供価値** | 脳 × 身体トレーニング（判断力、認知力、反応速度、チーム連携） |
| **サービス** | チームトレーニング、認知トレーニング、判断トレーニング |

### 3-5. 高齢者（Senior）

| 項目 | 内容 |
|------|------|
| **目的** | 健康寿命の延伸 |
| **提供価値** | 脳の活性化、身体機能維持、社会参加 |
| **サービス** | 脳活プログラム、運動プログラム、コミュニティ活動 |

## 5. キャラクターIP — LUMO（ルモ）

**概要**: 知恵と可能性を象徴するフクロウ。

**役割**: 可能性に光を当てるガイド。

**コンセプト**:
世界にはまだ開いていない、たくさんの可能性の扉がある。
LUMOはその鍵を見つける旅をしている。
子どもたちと一緒に、遊びながら可能性を見つけていく。

## 6. 事業戦略

UNLOCKは複数の市場を同時に小さく検証し、成果が出る領域にリソースを集中する。

### 初期戦略: 小規模実験（Lean Strategy）

| 実験領域 | 方法 |
|----------|------|
| スポーツ | 小さく試す → データを取る → 成功領域に集中 |
| 法人 | 同上 |
| 子ども | 同上 |
| プロフェッショナル | 同上 |
| 高齢者 | 同上 |

## 8. 10年ビジョン

| Phase | 期間 | テーマ | 主な取り組み |
|-------|------|--------|-------------|
| **1** | 0〜2年 | 実績づくり | スポーツ領域で導入、UNLOCK Method検証 |
| **2** | 2〜4年 | メソッド体系化 | UNLOCK Training、指導者育成 |
| **3** | 3〜6年 | B2B展開 | 企業研修、プロフェッショナル |
| **4** | 5〜8年 | キッズブランド展開 | LUMO IP、教育コンテンツ |
| **5** | 7〜10年 | グローバル展開 | スポーツ、教育、企業、健康 |
